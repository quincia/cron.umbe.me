cron.umbe.me

Technical blog on DevOps/Cloud, Backend/API, Pimcore, Symfony, Laravel and REST API — published automatically three times a week in Italian and English.

How it works

The pipeline runs every Monday, Wednesday and Thursday at 9:00 UTC via GitHub Actions.

Monday and Wednesday — Trend Stream

🔍 TrendAgent   →  reads HackerNews, DEV.to and GitHub Trending
🎯 TopicAgent   →  picks the most relevant topic (DevOps, Backend)

Thursday — PHP Stream

🎯 TopicAgent   →  rotates across Pimcore, Symfony, Laravel, REST/API

Both streams then:

✍️  WriterAgent  →  writes 2000+ words (IT and EN separately)
✏️  EditorAgent  →  reviews technical accuracy and flow
📌 SEOAgent     →  optimizes title, description, tags
🚀 Deploy       →  Hugo build → GitHub Pages

Each post includes generation time and token count.

Stack

  • Hugo + Minima theme
  • GitHub Actions for scheduling and CI/CD
  • Claude API (Anthropic, claude-opus-4-5)
  • GitHub Pages for hosting
