Event-Driven Architecture in Pimcore: Implementing CQRS and Event Sourcing for Scalable Enterprise Applications

Pimcore’s default architecture works well for small to medium projects. But when you’re dealing with millions of DataObjects, complex approval workflows, and the need for complete audit trails, the monolithic approach starts showing cracks. Batch imports that lock your database, versioning that consumes terabytes of storage, and APIs that can’t scale reads independently from writes—these problems compound as your data grows.

This article presents a battle-tested approach to implementing Event-Driven Architecture in Pimcore. We’ll decouple batch operations using Symfony Messenger, implement Event Sourcing for granular change tracking, apply CQRS to separate read and write workloads, and handle eventual consistency in multi-tenant scenarios. The patterns here come from production systems handling 50+ million DataObjects with sub-second API response times.

Prerequisites

Before implementing these patterns, ensure you have:

  • Pimcore 11.x with Symfony 6.4+
  • RabbitMQ 3.12+ or Redis 7.0+ for message transport
  • Elasticsearch 8.x for read-optimized projections
  • MySQL 8.0+ with appropriate isolation levels configured
  • Familiarity with Symfony Messenger and Doctrine event system
  • PHP 8.2+ with the php-amqp or predis extension

Your development environment should have Docker Compose set up with the required services. Production deployments need clustered message brokers and proper failover mechanisms.

Architecture and Key Concepts

Event Store: An append-only log that captures every state change as an immutable event. This replaces Pimcore’s native versioning with a more granular, queryable history.

CQRS (Command Query Responsibility Segregation): Write operations go through command handlers that validate business rules and emit events. Read operations query optimized projections instead of the normalized database.

Eventual Consistency: The read models eventually reflect write model changes, typically within milliseconds. Your application must handle the brief window where data might be stale.

<?php
// src/Message/Handler/UpdateDataObjectHandler.php

declare(strict_types=1);

namespace App\Message\Handler;

use App\Exception\OptimisticLockException;
use App\Message\Command\UpdateDataObjectCommand;
use App\Message\Event\DataObjectUpdatedEvent;
use App\Service\EventStore\EventStoreInterface;
use App\Service\Locking\OptimisticLockManager;
use Pimcore\Model\DataObject\AbstractObject;
use Pimcore\Model\DataObject\Concrete;
use Symfony\Component\Messenger\Attribute\AsMessageHandler;
use Symfony\Component\Messenger\MessageBusInterface;

#[AsMessageHandler]
final readonly class UpdateDataObjectHandler
{
    public function __construct(
        private EventStoreInterface $eventStore,
        private MessageBusInterface $eventBus,
        private OptimisticLockManager $lockManager,
    ) {
    }

    public function __invoke(UpdateDataObjectCommand $command): void
    {
        $object = AbstractObject::getById($command->objectId);
        
        if (!$object instanceof Concrete) {
            throw new \InvalidArgumentException(
                sprintf('Object %d not found or not a Concrete object', $command->objectId)
            );
        }

        // Verify expected version for optimistic locking
        if ($command->expectedVersion !== null) {
            $currentVersion = $this->eventStore->getCurrentVersion($command->objectId);
            
            if ($currentVersion !== $command->expectedVersion) {
                throw new OptimisticLockException(
                    sprintf(
                        'Version conflict: expected %d, found %d',
                        $command->expectedVersion,
                        $currentVersion
                    )
                );
            }
        }

        // Capture state before changes
        $previousState = $this->captureState($object);

        // Apply changes through Pimcore's API
        foreach ($command->changes as $field => $value) {
            $setter = 'set' . ucfirst($field);
            if (method_exists($object, $setter)) {
                $object->$setter($value);
            }
        }

        // Save with user action disabled to prevent recursive events
        $object->setOmitMandatoryCheck(true);
        $object->save();

        // Calculate actual changes (what really changed)
        $newState = $this->captureState($object);
        $actualChanges = $this->computeDiff($previousState, $newState);

        if (empty($actualChanges)) {
            return; // No actual changes, skip event
        }

        // Store event in event store
        $event = DataObjectUpdatedEvent::create(
            $command->objectId,
            $command->objectClass,
            $actualChanges,
            $previousState,
            $newState,
            $command->metadata,
        );

        $this->eventStore->append($event);

        // Dispatch for projections
        $this->eventBus->dispatch($event);
    }

    private function captureState(Concrete $object): array
    {
        $state = [];
        $class = $object->getClass();

        foreach ($class->getFieldDefinitions() as $key => $fd) {
            $getter = 'get' . ucfirst($key);
            if (method_exists($object, $getter)) {
                $value = $object->$getter();
                $state[$key] = $this->serializeValue($value);
            }
        }

        return $state;
    }

    private function serializeValue(mixed $value): mixed
    {
        if ($value instanceof \DateTimeInterface) {
            return $value->format('c');
        }

        if (is_object($value) && method_exists($value, 'getId')) {
            return ['_type' => get_class($value), '_id' => $value->getId()];
        }

        if (is_array($value)) {
            return array_map([$this, 'serializeValue'], $value);
        }

        return $value;
    }

    private function computeDiff(array $previous, array $new): array
    {
        $diff = [];

        foreach ($new as $key => $value) {
            if (!array_key_exists($key, $previous) || $previous[$key] !== $value) {
                $diff[$key] = [
                    'old' => $previous[$key] ?? null,
                    'new' => $value,
                ];
            }
        }

        return $diff;
    }
}

namespace App\Service\EventStore;

use App\Message\Event\DomainEvent;

interface EventStoreInterface
{
    /**
     * Append an event to the store.
     */
    public function append(DomainEvent $event): void;

    /**
     * Get all events for an aggregate.
     * 
     * @return iterable<DomainEvent>

    /**
     * Get current version number for optimistic locking.
     */
    public function getCurrentVersion(int $aggregateId): int;

    /**
     * Replay events to rebuild state at a specific point in time.
     * 
     * @return iterable<DomainEvent>

    /**
     * Get events across all aggregates for projection rebuilding.
     * 
     * @return iterable<DomainEvent>
     */
    public function getAllEventsSince(\DateTimeInterface $since, int $limit = 1000): iterable;
}

<?php
// src/Service/EventStore/MySQLEventStore.php

declare(strict_types=1);

namespace App\Service\EventStore;

use App\Message\Event\DomainEvent;
use Doctrine\DBAL\Connection;
use Doctrine\DBAL\Types\Types;

final class MySQLEventStore implements EventStoreInterface
{
    private const TABLE_NAME = 'app_event_store';

    public function __construct(
        private readonly Connection $connection,
    ) {
    }

    public function append(DomainEvent $event): void
    {
        $this->connection->transactional(function (Connection $conn) use ($event): void {
            // Get next version with row-level locking
            $currentVersion = (int) $conn->fetchOne(
                'SELECT COALESCE(MAX(version), 0) FROM ' . self::TABLE_NAME . 
                ' WHERE aggregate_id = ? FOR UPDATE',
                [$event->aggregateId],
            );

            $nextVersion = $currentVersion + 1;

            $conn->insert(self::TABLE_NAME, [
                'event_id' => $event->eventId->toRfc4122(),
                'aggregate_id' => $event->aggregateId,
                'aggregate_type' => $event->aggregateType,
                'event_type' => get_class($event),
                'version' => $nextVersion,
                'payload' => json_encode($event->toArray(), JSON_THROW_ON_ERROR),
                'metadata' => json_encode($event->metadata, JSON_THROW_ON_ERROR),
                'occurred_at' => $event->occurredAt,
                'tenant_id' => $event->metadata['tenant_id'] ?? null,
            ], [
                'occurred_at' => Types::DATETIME_IMMUTABLE,
            ]);
        });
    }

    public function getEvents(int $aggregateId, ?int $fromVersion = null): iterable
    {
        $qb = $this->connection->createQueryBuilder()
            ->select('*')
            ->from(self::TABLE_NAME)
            ->where('aggregate_id = :aggregate_id')
            ->setParameter('aggregate_id', $aggregateId)
            ->orderBy('version', 'ASC');

        if ($fromVersion !== null) {
            $qb->andWhere('version > :from_version')
               ->setParameter('from_version', $fromVersion);
        }

        $result = $qb->executeQuery();

        while ($row = $result->fetchAssociative()) {
            yield $this->hydrateEvent($row);
        }
    }

    public function getCurrentVersion(int $aggregateId): int
    {
        return (int) $this->connection->fetchOne(
            'SELECT COALESCE(MAX(version), 0) FROM ' . self::TABLE_NAME . 
            ' WHERE aggregate_id = ?',
            [$aggregateId],
        );
    }

    public function getEventsUntil(int $aggregateId, \DateTimeInterface $until): iterable
    {
        $result = $this->connection->executeQuery(
            'SELECT * FROM ' . self::TABLE_NAME . 
            ' WHERE aggregate_id = ? AND occurred_at <= ? ORDER BY version ASC',
            [$aggregateId, $until],
            [Types::INTEGER, Types::DATETIME_IMMUTABLE],
        );

        while ($row = $result->fetchAssociative()) {
            yield $this->hydrateEvent($row);
        }
    }

    public function getAllEventsSince(\DateTimeInterface $since, int $limit = 1000): iterable
    {
        $result = $this->connection->executeQuery(
            'SELECT * FROM ' . self::TABLE_NAME . 
            ' WHERE occurred_at > ? ORDER BY occurred_at ASC, version ASC LIMIT ?',
            [$since, $limit],
            [Types::DATETIME_IMMUTABLE, Types::INTEGER],
        );

        while ($row = $result->fetchAssociative()) {
            yield $this->hydrateEvent($row);
        }
    }

    private function hydrateEvent(array $row): DomainEvent
    {
        $eventClass = $row['event_type'];
        $payload = json_decode($row['payload'], true, 512, JSON_THROW_ON_ERROR);

        if (!class_exists($eventClass) || !is_subclass_of($eventClass, DomainEvent::class)) {
            throw new \RuntimeException(sprintf('Unknown event type: %s', $eventClass));
        }

        return $eventClass::fromArray($payload);
    }
}
