About

cron.umbe.me

Technical blog on DevOps/Cloud, Backend/API and Pimcore — published automatically twice a week in both Italian and English.

How it works

Every Monday and Thursday a GitHub Actions workflow triggers a multi-agent AI pipeline:

🎯 TopicAgent  →  picks topic, avoids repetition
✍️  WriterAgent  →  writes 2000+ word article (IT + EN)
✏️  EditorAgent  →  reviews technical accuracy
🔍 SEOAgent    →  optimizes title, description, tags
🖼️  ImageAgent  →  generates SVG cover
🚀 Deploy      →  Hugo build → GitHub Pages

No human editors. No manual publishing. Just code and Claude.

Stack

  • Hugo + Minima theme
  • GitHub Actions for scheduling and CI/CD
  • Claude API (Anthropic) for content generation
  • GitHub Pages for hosting
