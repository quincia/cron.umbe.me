Blog tecnico su DevOps/Cloud, Backend/API, Pimcore, Symfony, Laravel e REST API — pubblicato automaticamente tre volte a settimana in italiano e in inglese.

Come funziona

La pipeline si avvia ogni lunedì, mercoledì e giovedì alle 9:00 UTC tramite GitHub Actions.

Lunedì e mercoledì — Stream Trend

🔍 TrendAgent   →  analizza HackerNews, DEV.to e GitHub Trending
🎯 TopicAgent   →  sceglie il topic più rilevante (DevOps, Backend)

Giovedì — Stream PHP

🎯 TopicAgent   →  rotazione su Pimcore, Symfony, Laravel, REST/API

Entrambi gli stream poi:

✍️  WriterAgent  →  scrive 2000+ parole (IT e EN separatamente)
✏️  EditorAgent  →  revisiona accuratezza tecnica e flusso
📌 SEOAgent     →  ottimizza titolo, description e tag
🚀 Deploy       →  build Hugo → GitHub Pages

Ogni post riporta il tempo di generazione e i token consumati.

Stack

  • Hugo + tema Minima
  • GitHub Actions per scheduling e CI/CD
  • Claude API (Anthropic, claude-opus-4-5)
  • GitHub Pages per l’hosting
