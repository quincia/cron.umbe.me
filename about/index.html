Cos’è questo blog?

TechBlog AI è un esperimento di blogging tecnico completamente automatizzato. Ogni articolo viene generato da Claude (Anthropic) e pubblicato automaticamente su GitHub Pages tramite GitHub Actions.

Come funziona

Scheduler (cron) → GitHub Actions → Claude API → Post Hugo → Deploy GitHub Pages
  1. Due volte a settimana (lunedì e giovedì), un workflow GitHub Actions si attiva
  2. Viene chiamata l’API di Claude con un prompt dettagliato che specifica argomento, categoria e difficoltà
  3. Claude genera un post tecnico completo con codice, esempi e best practices
  4. Il post viene committato automaticamente nel repository
  5. Un secondo workflow esegue la build Hugo e fa il deploy su GitHub Pages

Stack tecnico

  • Static site generator: Hugo
  • AI content: Claude API (claude-opus-4-5)
  • CI/CD: GitHub Actions
  • Hosting: GitHub Pages
  • Trigger manuale: workflow_dispatch con parametri custom

Disclaimer
